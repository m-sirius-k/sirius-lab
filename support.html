How can we help?

Setup guides, troubleshooting, and answers to common questions about Orchestra.

Setup Guide
1

Install Orchestra from Chrome Web Store

Click "Add to Chrome" on the store page. Chrome will ask for permission — click "Add extension" to confirm.

2

Open claude.ai in Chrome

Orchestra only works on claude.ai. Make sure you are logged in to your Claude account before continuing.

3

Start a conversation

Send any message on claude.ai. Orchestra begins capturing automatically — no button to press, no configuration needed.

4

Open the Orchestra popup

Click the Orchestra icon in your Chrome toolbar (top-right). You will see your saved messages, search bar, and export buttons.

5

Export anytime

Click ↓ CSV or ↓ JSON to download your full conversation history. No account required. No limits.

Troubleshooting
ERROR Messages are not being saved

Orchestra is not capturing new messages from claude.ai.

  • The extension may not have loaded on this tab.
  • You may be on a page other than claude.ai.
  • Chrome may have disabled the extension after an update.
FIX Go to chrome://extensions → find Orchestra → toggle OFF then ON → reload claude.ai and try again.
WARNING Popup shows 0 messages after install

Orchestra only saves messages from the moment it is installed. Past conversations are not retroactively captured.

EXPECTED BEHAVIOR Send at least one message on claude.ai after installing. The counter will update within seconds.
WARNING Export CSV / JSON button does nothing

The download may be blocked by Chrome's download settings, or there is no data to export yet.

  • Check that Chrome is not blocking downloads from extensions.
  • Make sure at least one message has been saved first.
FIX Go to Chrome Settings → Downloads → make sure "Ask where to save each file" is not blocking automatic downloads.
ERROR Extension icon is grayed out / inactive

Orchestra is only active on claude.ai pages. On all other sites the icon will appear grayed out — this is expected behavior.

EXPECTED BEHAVIOR Navigate to claude.ai. The icon will become active automatically.
INFO Data disappeared after reinstalling

Orchestra stores all data in your browser's local IndexedDB. Uninstalling the extension permanently deletes all saved data — this is by design for privacy.

RECOMMENDATION Export your data as CSV or JSON regularly as a backup before uninstalling or updating Chrome.
WARNING Search returns no results

Search is case-sensitive on some builds, or the keyword may not exactly match saved content.

  • Try shorter keywords (e.g. "python" instead of "Python script").
  • Switch between ALL / YOU / AI tabs to narrow the scope.
TIP Use the "All time" date filter and "Newest" sort to see all available messages first.
Frequently Asked Questions
Does Orchestra send my data anywhere?
No. Orchestra has no server. All data is stored exclusively in your browser's IndexedDB. It is physically impossible for Orchestra to transmit your data externally.
Does Orchestra work with ChatGPT or Gemini?
The current free version is designed specifically for claude.ai. Support for other AI platforms may be added in future versions.
Do I need an API key?
No. Orchestra uses your existing browser session — the account you already have open in Chrome. No API key purchase is required.
How much data can Orchestra store?
Storage is limited by your browser's IndexedDB quota, which is typically several hundred megabytes. For most users this means thousands of conversations before any limit is reached.
Can I use Orchestra on multiple computers?
Data is stored locally per browser, so it does not sync across devices. Export your data as JSON and import it manually if needed.
How do I completely remove Orchestra and all its data?
Go to chrome://extensions → find Orchestra → click "Remove". All data stored by the extension is automatically deleted by Chrome upon removal.
